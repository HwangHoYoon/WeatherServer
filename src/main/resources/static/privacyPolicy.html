개인정보처리방침

'레디우산(이하 서비스)'은 통신비밀보호법, 전기통신사업법, 정보통신망 이용촉진 및 정보보호 등에 관한 법률 등 정보통신서비스 제공자가 준수하여야 할 관련 법령상의 개인정보보호 규정을 준수하며, 관련 법령에 의거한 개인정보취급방침을 정하여 이용자 권익 보호에 최선을 다하고 있습니다.

제 1조 (개인정보의 수집 및 이용 목적)

개인정보는 생존하는 개인에 관한 정보로서 서비스 이용자를 식별할 수 있는 정보(해당 정보만으로는 특정 개인을 식별할 수 없더라도 다른 정보와 용이하게 결합하여 식별할 수 있는 것을 포함)를 말합니다. 서비스가 수집한 개인정보는 서비스의 기본 기능 제공을 목적으로 활용합니다.

제 2조 (수집되는 데이터 유형)

계정 데이터

계정 데이터는 회원에게서만 수집됩니다.

사용자가 카카오 또는 애플 계정으로 회원 가입하는 과정에서 서비스는 아래 정보를 수집합니다.

  • 이름
    • 당사가 수집하는 이름 정보는 사용자의 실제 성명과 다를 수 있습니다. 사용자는 회원 가입 후 이름을 편집할 수 있습니다.
  • 이메일 주소
    • 사용자가 애플 계정으로 가입하는 경우, 당사는 사용자가 명시적으로 동의한 경우에 한해서만 실제 이메일 주소를 수집할 수 있습니다. 사용자가 동의하지 않은 경우, 당사는 애플에서 임의로 생성한 랜덤 이메일 주소를 수집합니다.

본 약관의 최종 업데이트 일자 기준으로, 계정 데이터는 아래 정보를 포함하지 않습니다.:

  • 결제 정보
  • 사용 데이터

서비스에 회원 가입하는 사용자는 카카오 또는 애플 로그인의 개인 정보 취급 방침의 적용을 받습니다.

애플 로그인의 개인 정보 취급 방침 및 정책에 대한 자세한 내용은 해당 개인 정보 취급 방침 페이지 https://www.apple.com/legal/privacy/data/en/sign-in-with-apple/를 참조하십시오.

카카오 로그인의 개인 정보 취급 방침 및 정책에 대한 자세한 내용은 해당 개인 정보 취급 방침 페이지

https://www.kakao.com/ko/privacy를 참조하십시오.

개인 데이터

서비스는 서비스를 사용하는 동안 사용자에게 연락하거나 신원을 확인하는 데 사용할 수 있는 특정 개인 식별 가능 정보를 제공하도록 요청할 수 있습니다. 개인 식별 가능 정보는 다음을 포함할 수 있지만 이에 국한되지는 않습니다.

  • 계정 데이터 중 이름
  • 이메일 주소
  • 국가, 주소, 주, 도, 시, 우편번호
  • 사용 데이터
  • 광고 식별자(ADID)
    • Android 기기의 Google Advertising ID(GAID)
    • iOS 기기의 광고주 식별자(IDFA)

사용 데이터

서비스 이용 시 사용 데이터는 자동으로 수집됩니다.

사용 데이터에는 사용자 기기의 인터넷 프로토콜 주소(예: IP 주소), 브라우저 유형, 브라우저 버전, 사용자가 방문하는 서비스 페이지, 사용자의 방문 시간 및 날짜, 해당 페이지에서 보낸 시간, 고유 장치 식별자 및 기타 진단 데이터와 같은 정보가 포함될 수 있습니다.

사용자가 모바일 기기를 통해 또는 모바일 기기를 통한 서비스에 액세스할 때, 당사는 사용자가 사용하는 모바일 기기의 유형, 사용자의 모바일 기기 고유 ID, 사용자의 모바일 기기 IP 주소, 사용자의 모바일 운영체제, 사용자가 사용하는 모바일 인터넷 브라우저의 유형, 고유한 장치 식별자 및 기타 진단 데이터를 포함하되 이에 국한되지 않는 특정 정보를 자동으로 수집할 수 있습니다.

또한, 당사는 사용자가 당사 서비스를 방문할 때마다 또는 사용자가 모바일 기기를 통해 서비스에 액세스할 때 사용자의 브라우저가 보내는 정보를 수집할 수 있습니다.

  • 사용자에게 제공: 사용자가 그러한 정보를 수신하지 않기로 선택하지 않는 한 사용자가 이미 구매했거나 문의한 것과 유사한 기타 상품, 서비스 및 이벤트에 대한 뉴스, 특별 제안 및 일반 정보를 사용자에게 제공하기 위해.
  • 사용자의 요청 관리: 서비스에 대한 사용자의 요청에 주의를 기울이고 관리하기 위해
